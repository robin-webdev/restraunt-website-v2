# Gongu Restaurant Website Demo

Hi, I'm Robin. I built this restaurant website demo as a modern frontend project to showcase how a food and hospitality brand can have a polished, interactive, and mobile-friendly online presence.

This demo focuses on elegant presentation, smooth user experience, and responsive design. It includes animated sections, a featured dishes showcase, menu previews, testimonials, reservation call-to-actions, location details, and multilingual support.

## About The Project

Gongu Restaurant is a single-page restaurant website demo designed to feel premium and visually engaging. The goal was to create a realistic restaurant landing page that looks great across devices while keeping the structure clean and scalable for future improvements.

## Tech Stack

This project was built with:

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Framer Motion
- Wouter
- Radix UI
- Lucide React
- ESLint

## Features

- Responsive restaurant landing page layout
- Hero section with strong visual branding
- Animated content sections using Framer Motion
- Featured dishes cards with call-to-action buttons
- Interactive menu preview with category switching
- About section and brand storytelling
- Gallery section for restaurant ambiance and food visuals
- Customer testimonials section
- Reservation call-to-action area
- Contact form in the footer
- Location and opening hours section
- Mobile bottom navigation bar for quick actions
- Multilingual support

## Getting Started

To run the project locally:

```bash
npm install
npm run dev
```

To create a production build:

```bash
npm run build
```

## Purpose

I created this demo to practice and showcase modern frontend development for restaurant and hospitality websites. It can also serve as a starting point for a real client project, portfolio piece, or UI inspiration.
